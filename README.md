# ✈️ Flight Information Android UI

A modern Android UI built using **Skyscanner Backpack components** to display flight details including departure and arrival information.

---

## 📱 Features

* Flight information card
* Departure and arrival sections
* Clean card-based layout
* Uses **ConstraintLayout + Backpack UI**
* Material-style design

---

## 🛠️ Tech Stack

* Java / Kotlin (Android)
* XML Layouts
* ConstraintLayout
* Skyscanner Backpack UI

---

## 📸 UI Preview

* Flight number and seat details
* Departure (LHR - London Heathrow)
* Arrival (JFK - New York)

---

## 📂 Project Structure

```
app/
 └── src/main/
      ├── java/
      ├── res/layout/
      │     └── activity_main.xml
      └── res/values/
```

---

## 🚀 How to Run

1. Clone the repo
2. Open in Android Studio
3. Sync Gradle
4. Run on emulator/device

---

## 💡 Notes

* Uses `BpkCardView` and `BpkText`
* Designed for clean travel UI experience
* Easily extendable for full booking apps
